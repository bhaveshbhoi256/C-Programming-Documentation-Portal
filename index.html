  1. Introduction

    • Overview: C is a powerful, general-purpose programming language, renowned for system programming, performance, and control over memory. Created by Dennis Ritchie in the 1970s, C has influenced many modern programming languages.

    • Applications: Operating systems, embedded systems, compilers, and performance-intensive applications.

    • Audience: This documentation is for beginners, intermediate programmers, and advanced users looking to understand C's features in-depth.


  2. Program Structure

    • Header Files: Libraries like stdio.h, stdlib.h for input/output, memory, and more.

    • Main Function: The starting point of execution.

    • Syntax Overview: Statements end with ;, code blocks use {}, and comments use // or /* ... */.

    • Syntax

      #include < stdio.h>
      int main() {
      printf("Hello, World!\n");
      return 0;
      }


  3. Data Types and Variables

    • Primitive Types: int, char, float, double

    • Modifiers: short, long, signed, unsigned

    • Derived Types: Arrays, Pointers, Structures, Unions, Enums

    • Declaration and Initialization:

      Example:

      int num = 10;
      char letter = 'A';


  4. Operators

    • Arithmetic Operators: +, -, *, /, %

    • Relational Operators: ==, !=, >, <, >=, <=

    • Logical Operators: &&, ||, !

    • Bitwise Operators: &, |, ^, ~, <<, >>

    • Assignment Operators: =, +=, -=, etc.

    • Unary Operators: ++, --

  5. Control Flow

    • Conditionals: if, if-else, switch

    • Loops: for, while, do-while

      Example:

      for (int i = 0; i < 10; i++) {
      printf("%d\n", i);
      }

  6. Functions

    • Declaration and Definition: Functions are declared with a return type and parameters.

    • Function Prototypes: Allow functions to be declared before they’re used.

    • Definition:Functions have a return type and parameters.

      Example:

      int add(int a, int b) {
      return a + b;
      }

  7. Memory management

    • Pointers: int *p for memory addresses, malloc, free for dynamic memory.

      Example:

      int *ptr = (int*)malloc(sizeof(int) * 5);
      free(ptr);

  8. Structures and Unions

    • Structures: Group variables (e.g., struct Point { int x; int y; };).

    • Unions: Share memory across members.

  9. File I/O

    • Basic Functions: fopen, fprintf, fclose

      Example:

      FILE *file = fopen("file.txt", "w");
      fprintf(file, "Hello, File!\n");
      fclose(file);

  10. Error Handling

    • Return Codes: Functions typically return 0 for success, non-zero for failure.

    • Error Reporting: perror for I/O errors

  11. Standard Library

    • Categories: < stdio.h> for I/O, < stdlib.h> for memory management, < string.h> for strings, < math.h> for math functions.
